# Basic Calculator (Addition & Subtraction)

This is a very simple JavaScript calculator that can add or subtract two numbers entered by the user.

It is one of my first JavaScript exercises and part of my learning journey as a future Full-Stack Developer.

---

## üîß What this project does

- Asks the user for two numbers (using `prompt`)
- Asks the user which operation they want:
  - `addition`
  - `subtraction`
- Uses a function to:
  - take the two numbers and the operation type as parameters
  - calculate the correct result
  - return the result
- Prints the result to the browser console
- Shows an error message if the operation type is invalid

## ‚ñ∂Ô∏è How to run this project

1. Open any modern browser (Chrome, Firefox, Edge, etc.).
2. Open the **Developer Tools**:
   - Windows: `F12` or `Ctrl + Shift + I`
   - Mac: `Cmd + Opt + I`
3. Go to the **Console** tab.
4. Copy the contents of `basic-calculator.js` and paste it into the console.
5. Press `Enter`.
6. Follow the prompts:
   - Enter the first number
   - Enter the second number
   - Type the operation: `addition` or `subtraction`
7. The result will be printed in the console.
